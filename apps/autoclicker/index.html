AutoClicker & Macro

A small macOS desktop tool that does two things: clicks the mouse on a timer, and records / replays your mouse and keyboard as a macro.

  • Python
  • tkinter
  • pynput
  • Desktop
  • Automation

What it does

The auto-clicker fires the mouse at a configurable interval — left, right or middle button, single or double click, running forever or stopping after a set number of clicks. The macro recorder captures real mouse clicks, scrolls and key presses with their timing, then replays them at an adjustable speed and loop count. Macros save to JSON so you can reload them later. The clicking and macro engines are kept separate from the GUI, so the automation logic stays reusable and testable on its own.

Hotkeys

F6Start / stop the auto clicker
F7Start / stop recording a macro
F8Play / stop the recorded macro

Run it locally

git clone https://github.com/lxreyes/auto.git
cd auto
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py

Requires Python 3 on macOS. This is a desktop app, so it runs on your machine rather than in the browser.

macOS permissions

macOS blocks input control until you allow it. Grant the app that launches Python (your Terminal or VS Code) both Accessibility and Input Monitoring under System Settings → Privacy & Security, then quit and reopen it. If clicks don’t register or recordings stay empty, that’s almost always the reason.
