SKY SIM

2026

A 3D flight simulator. Take off over procedurally generated terrain — hills, water, forests and drifting clouds — with an arcade flight model: throttle, pitch / roll / yaw with coordinated banking turns, lift, drag, gravity, and stall and landing / crash detection. Switch between chase and cockpit cameras, read a full glass HUD with airspeed, altitude and an attitude indicator, all set to procedural engine and wind audio. One self-contained HTML file on Three.js, no build step.

CALM

2026

A tiny calm corner of the internet. Pick an ambient sound — rain on a metal roof, a fireplace, the ocean, wind, forest birds or night crickets — and a matching pixel scene drifts past in a first-person walk. Every sound is synthesized live with the Web Audio API and every scene is drawn pixel-by-pixel — no audio files, no images, no tracking. Runs completely offline.

AUTOCLICKER & MACRO

2026

A macOS desktop utility. The auto-clicker fires the mouse at a configurable interval — left / right / middle, single or double, forever or for a set count. The macro recorder captures real mouse and keyboard input with timing, then replays it at adjustable speed and loop count, and saves to JSON. Global hotkeys (F6 / F7 / F8) drive it even when the window is hidden. Built with Python + tkinter + pynput, with the clicking and macro engines kept separate from the UI.

  • Python
  • tkinter
  • pynput
  • Desktop
