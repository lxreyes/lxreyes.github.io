AUTOCLICKER & MACRO

2026

A macOS desktop utility. The auto-clicker fires the mouse at a configurable interval — left / right / middle, single or double, forever or for a set count. The macro recorder captures real mouse and keyboard input with timing, then replays it at adjustable speed and loop count, and saves to JSON. Global hotkeys (F6 / F7 / F8) drive it even when the window is hidden. Built with Python + tkinter + pynput, with the clicking and macro engines kept separate from the UI.

  • tkinter
